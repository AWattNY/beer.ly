# beer.ly
Delivery beer flights. Beerly is an app that allows users to have beer flights be delivered to them.

### Tech Stack
1. Node / Express
1. React 
1. Mongoose/MongoDB
1. Google Maps API
1. StripeJS
1. Material UI
1. BreweryDB

## Quick Start


### Fork/Clone
```
git clone https://github.com/ntoung/beer.ly.git
```

### Install Dependencies
```
npm install
```

### Run
```
npm start
```
